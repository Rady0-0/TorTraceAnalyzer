Sample Inputs
=============

This folder contains small demo files that are safe to use for a first run of the tool.

Files included
- clean_input.txt: a low-signal example
- tor_demo_system.txt: system/application-style Tor indicators
- tor_demo_network.log: network-style Tor indicators
- tor_demo_memory.raw: memory-style Tor indicators
- tor_demo.pcap: a tiny Tor-port packet capture sample
- tor_demo.pcapng: the same idea in pcapng format

Purpose
- confirm that the EXE launches,
- confirm detections appear in the Dashboard and layer tabs,
- confirm `.pcap` and `.pcapng` analysis are routed correctly.

These are demonstration files only and not real forensic evidence.
